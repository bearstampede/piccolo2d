This directory contains verious tools that easy the
maintenance of the piccolo2d website.

Usually they must be started with the site's root as current
directory.

Each file starts with a brief description of it's purpose.

The most important are:

svn-mime-types.sh: Set the correnct mimetypes of all files (requires find, xargs)

giftopng.sh: convert a gif image to png (requires netpbm, replace)

validate.sh: validate xhtml (requires cat, xmllint, trang)

inject_navigation.rb: inject navigation into a html file (requires ruby)
